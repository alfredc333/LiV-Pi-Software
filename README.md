<h1>LiV Pi</h1>

Raspberry Pi Indoor Air Quality Monitor.

Final Submission. 

<h3>Raspbian OS. Python 2.7 </h3> 

<h4>CO2. Temperature. Humidity. Air Pressure.</h4>

<h3>Sensors:</h3>

- BME180 - I2C

- DHT22 - 1 wire proprietary

- Senseair K30 - serial

- RTC DS3231 - I2C

- LCD1602 - I2C
